\section{Rank 2 Algebras}

\begin{frame}{Classification of Rank 2 Root Systems}
    Using the allowed angles, we can geometrically construct all possible rank 2 root systems (algebras with a 2-dimensional Cartan subalgebra).

    \vspace{0.3cm}
    There are exactly four mathematical solutions:
    \begin{enumerate}
        \item $A_1 \oplus A_1$ (e.g., $SU(2) \times SU(2)$)
        \item $A_2$ (e.g., $SU(3)$, the Eightfold Way flavor symmetry)
        \item $B_2 \cong C_2$ (e.g., $SO(5)$)
        \item $G_2$ (Exceptional algebra)
    \end{enumerate}
